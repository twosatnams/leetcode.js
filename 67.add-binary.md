## Question

Given two binary strings, return their sum (also a binary string).

For example,
a = "11"
b = "1"
Return "100".

-----------------------------------------------------------------------------

1. Make sure that both strings are equal length.
2. Keep a carry variable.
3. Simulate addition by hand. 3 = 11, 2 = 10 etc.

-----------------------------------------------------------------------------

```
def add_binary(a, b)
    # return (a.to_i(2) + b.to_i(2)).to_s(2)

    # lets make sure they are the same length
    difference = (a.length - b.length).abs
    a.length <= b.length ? a = ("0" * difference) + a : b = ("0" * difference) + b
    # puts "Before: " + a + " : " + b

    idx = a.length - 1
    carry = 0
    result = ""
    while idx >= 0
        first = a[idx]
        second = b[idx]
        sum = first.to_i + second.to_i + carry
        # puts "Sum: " + sum.to_s
        if sum == 3
            result = "1" + result
            carry = 1
        elsif sum == 2
            result = "0" + result
            carry = 1
        elsif sum == 1
            result = "1" + result
            carry = 0
        elsif sum == 0
            result = "0" + result
            carry = 0
        end
        # puts "Result: " + result

        idx = idx - 1
    end

    result = "1" + result if carry == 1

    return result
end
```
